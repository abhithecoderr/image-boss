**Purpose**:

A slider for result canvas in upscaling service which is dragged to compare the upscaled vs original image


**State**:

*position*: Percentage of where the divider sits horizontally.

*isDragging*: Tracks whether the user is currently holding the mouse down on the slider

*containerRef*: Points to outer div wrapper, to capture mouse position relative to canvas boundaries

*upRef*: Points to the canvas that shows upscaled image

*origRef*: Points to original canvas
