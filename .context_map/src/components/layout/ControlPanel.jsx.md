**Purpose**

Renders the control panel under workspace area for various services

**Imports**

**State extraction**

**renderServiceControls**

Renders controls as per the service selected

**getProcessAction() and getProcessLabel() functions**

Help decide the action and label for the process button respectively based on the mode of operation (single, batch)

**return block**

To render mode selector dropdown, service specific controls,  select/deselect and process button
